# 💻 Calculator App

A modern, responsive calculator built using HTML, CSS, and JavaScript with a clean UI and smooth user experience.

---

## 🚀 Features

- Animated gradient background  
- Glassmorphism UI design  
- Smooth button animations and ripple effect  
- Keyboard support  
- Error handling for invalid inputs  
- Responsive layout  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  

---

## 📂 Project Structure

```
calculator-app/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ Getting Started

### 1. Clone the repository
```
git clone https://github.com/rekha2756/calculator-app.git
```

### 2. Navigate to project folder
```
cd calculator-app
```

### 3. Run the project
Open `index.html` in your browser
